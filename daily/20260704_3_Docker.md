# 2026年07月04日　まとめ3.Docker(1)
## 1. Dockerfileを作成する
今回は、docer-practiceというプロジェクトが進行すると見なす。projects/docker-practice/直下にDockerfileを作成する(拡張子はない。大文字小文字を注意すること)。
1. 第一段階として、以下を記述する。
    ```Dockerfile
    FROM python3:13
    WORKDIR /app
    ```
   * 1行目はベースとなるコンテナイメージ(以下、イメージと呼ぶ)を指定している。今回はpython3:13。
   * 2行目はコンテナが動いた後に、ターミナル(bash)で、
     ```bash
     cd /app
     ```
    を自動で実行させる。
2. COPY命令を追加する。
    ```Dockerfile
    FROM python3:13
    WORKDIR /app
    COPY . .
    ```
    COPYの意味は
    ```Dockerfile
    COPY <コピー元> <コピー先>
    ```
    また、"."は現在のディレクトリ（カレントディレクトリ）を意味する。よって、
    * 1つ目の"."はホスト側：docker-practiceフォルダ
    * 2つ目の"."はコンテナ側のカレントディレクトリ(WORKDIR /appで指定した/app)という意味になる。
3. イメージをビルドするときに、一度だけ実行される命令を書く。今回はpythonの外部パッケージをpipコマンドでインストールしなければいけないので、次のように記述する
    ```Dockerfile
    FROM python:3.13
    WORKDIR /app
    COPY . .
    RUN pip install -r requirements.txt
    ```
    ただし、requirements.txtファイルの中身は
    ```
    fastapi
    uvicorn
    ```
    である。両方pythonのパッケージ名である。

4. ビルドしたイメージを走らせるときに、毎回実行される命令を書く。まずは、docker-practice/app/main.pyという構造になるように、appフォルダ、main.pyファイルを作成し、main.pyファイルは次のように書く(中身の詳細は別の機会に記す)。
    ```python
    from fastapi import FastAPI

    app = FastAPI()

    @app.get("/")
    def root():
        return {"message": "Hello Docker!"}
    ```
    次に、DockerFileにCMDを追加する。
    ```Dockerfile
    FROM python:3.13
    WORKDIR /app
    COPY . .
    RUN pip install -r requirements.txt
    CMD ["uvicorn", "app.main:app", "--host", "0.0.0.0", "--port", "8000"]
    ```
    CMDの各要素について説明しておく。
    * uvicorn : fastAPIアプリを動かすためのWebサーバー。実際には、fastAPI専用ではなく、ASGIアプリケーションを動かすWebサーバーである。
    * app.main:app : app/main.pyの中にあるappという変数を起動してください。という意味。
    * --host 0.0.0.0 : 外部からアクセスできるようにする
    * --post 8000 : 8000番ポートで待ち受ける
5. イメージをビルドして、走らせる。
    ```bash
    docker build -t docker-practice .
    docker run -d -p 8000:8000 docker-practice
    ```
    * 1行目は、docker-practiceというイメージ名で"."、即ちカレントディレクトリ(docker-practiceフォルダ。名前が被っててややこしい)をビルド対象にして、ビルドを行うという意味である。
    * 2行目は、docker-practiceという名前のイメージを走らせて、コンテナを作るという意味である。

    このコマンドを叩いた後、http://localhost:8000 にアクセスすると、
    ```JSON
    {"message":"Hello Docker!"}
    ```
    が表示されている。
    次回は、pythonについてもう少し詰めたい。
