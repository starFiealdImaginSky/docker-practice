# 2026年07月04日　まとめ1.環境構築

## 1. WSL(Ubuntu)のインストール・Docker環境構築

1. まずはPowerShellでインストールを行う。
   
    ```powershell
    wsl --install -d Ubuntu
    ```

    初回起動時に
    ```
    Enter new UNIX username:
    ```
    と聞かれるため、好きなユーザー名を設定する。
    さらに、パスワードを2回入力する（指示されるので入力すればよい）。


2. 次に、インストールしたUbuntuを開き、次の命令を叩く。

    ```bash
    sudo apt update
    sudo apt upgrade -y
    sudo apt install -y git curl unzip build-essential
    ```


3. 次に、DockerをUbuntuに結びつける。まずは、Windows側で  Docker Desktopをインストールする。その後、Docker Desktopを開いて、Setting→Resources→WSL Integrationから次の設定をONにする。

   * Enable integration with my default WSL distro
   * Ubuntu

    設定を保存してDockerを再起動する。

4. PowerShellで一度Ubuntuを終了して、再起動する。
5. Ubuntuで次のコマンドを叩き、実行できれば問題なく設定できている。
   ```bash
   docker --version
   docker compose version
   docker run hello-world
   ```

## 3. git,GitHubの初期設定
1. gitのインストールから行う。Ubuntuで、次のコマンドを叩く。
   ```bash
   sudo apt install git
   ```
   インストール出来たかは、次のコマンドをPoserShellで叩けばよい。
   ```bash
   git --version
   ```

2. 事前にGitHubでアカウントを登録しておく。その上で、UbuntuにてGitHubで使っている情報を設定する。
    ```bash
    git config --global user.name "GitHubのユーザー名"
    git config --global user.email "GitHubに登録しているメールアドレス"
    ```
    情報が設定できたかは、次のコマンドを叩けばよい。
    ```bash
    git config --global --list
    ```
3. ついでに次のコマンドを叩いておく。
    ```bash
    # デフォルトブランチの名前を main にする
    git config --global init.defaultBranch main

    # push時の挙動をシンプルに
    git config --global push.autoSetupRemote true

    # デフォルトエディタをVS Codeにする
    git config --global core.editor "code --wait"
    ```

4. Ubuntu上でSSH鍵を作成する。GitHubで使う場合、**Ed25519**方式がおすすめらしい。
   ```bash
   ssh-keygen -t ed25519 -C "GitHubに登録しているメールアドレス"
   ```
   上のコマンドを叩くと、次のように連続で聞かれるが、Enterを連打で良い。ただし、企業で作る場合は、パスフレーズを入力した方が良さそう。
   ```
   Enter file in which to save the key (/home/hoshino/.ssh/id_ed25519):
   ```
   ```
   Enter passphrase (empty for no passphrase):
   ```
   SSH鍵が作成できているか確認をしたい場合は次のコマンドを叩けばよい。
   ```bash
   ls ~/.ssh
   ```
5. GitHubに公開鍵を登録する。
   まずは、公開鍵(ざっくりUbuntuとGitHubを繋げるためのもの)の内容を表示し、コピーする。
   ```bash
   cat ~/.ssh/id_ed25519.pub
   ```
   これを、GitHub上のSetting→SSH and GPG keys→SSH keysからNew SSH keyボタンを押して登録すればよい。
